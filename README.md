## Stack used

- Angular `2.4.0`
- TypeScript `2.0.10`
- JavaScript `ES6`

## Getting started

#### Requirements

- `yarn` (or `npm`)

#### Installation

```sh
git clone https://github.com/jclerc/colibrique-client.git
cd colibrique-client
yarn # or `npm i`
yarn start # or `npm start`
```
